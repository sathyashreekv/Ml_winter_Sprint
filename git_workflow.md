# Daily Git Workflow

## After each day's work:

1. **Check status:**
   ```bash
   git status
   ```

2. **Add changes:**
   ```bash
   git add .
   ```

3. **Commit with meaningful message:**
   ```bash
   git commit -m "Day X: Topic covered - brief description"
   ```

4. **Push to GitHub:**
   ```bash
   git push
   ```

## Example commit messages:
- "Day 1: Linear algebra basics - vectors and matrices"
- "Day 2: NumPy fundamentals and array operations"
- "Day 3: Pandas data manipulation and cleaning"

## Branch strategy (optional):
- Create feature branches for major topics
- Merge to main when complete
